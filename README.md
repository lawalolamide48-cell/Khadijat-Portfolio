## Overview

A bold, creative personal portfolio built from scratch to showcase both administrative and full-stack development skills. Designed with a deep violet, electric coral, and cream colour palette — clean, professional, and memorable.

The portfolio is a static site requiring no frameworks, build tools, or servers — just open `index.html` in any browser.

---

## Features

- **Sticky navigation** with smooth scroll and mobile hamburger menu
- **Project carousel** — swipe or click through 4 featured projects, with touch and mouse drag support
- **Scroll-reveal animations** — sections fade in as you scroll
- **Profile photo** in the About section
- **Read More page** — links to a dedicated `about.html` with full background story
- **Contact section** with email, phone, WhatsApp, LinkedIn, and GitHub
- **Downloadable CV** direct from the hero section
- **Accessibility** — skip link, ARIA labels, keyboard navigation, reduced motion support
- **Fully responsive** — mobile, tablet, and desktop

## Getting Started

### View locally

1. Clone the repository:
```bash
git clone https://github.com/lawalolamide48-cell/Khadijat-Portfolio.git
```

2. Open in VS Code

3. Right-click `index.html` → **Open with Live Server**

4. Opens at `http://127.0.0.1:5500`

No npm install, no build step, no environment variables needed.

### Deploy updates

```bash
git add .
git commit -m "Your update message"
git push origin main
```

GitHub Pages updates automatically within 1–2 minutes.
